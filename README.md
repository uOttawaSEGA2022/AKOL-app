AKOL is a **food delivery** mobile application built with Java for Android that acts as a simplified, course-project version of Uber Eats. 
It allows users to browse restaurants, explore menus, and place delivery orders directly from their smartphones.

---

## Overview

AKOL is a personal/student project, focusing on core food ordering flows rather than full production complexity. 
The app was developed using Android Studio and Java.


- Practicing mobile app architecture and navigation in Android (activities/fragments, adapters, etc.).  
- Managing data models for restaurants, menus, and orders.  
- Implementing user flows similar to modern food delivery apps.

---

## Key Features

- Browse a list of restaurants with names, images, and basic info. 
- View restaurant menus with food items, descriptions, and prices.  
- Add items to a cart/order with quantity selection.
- Review order details before confirming. 
- Basic tracking of order status (e.g., pending / preparing / delivered).
- User authentication (login/register).  
- Order history for past orders.
